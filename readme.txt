System Requirements:
 - Node & npm
 
*********************Before running the game*******************
Game uses dictionary file commonly located at 
/usr/share/dict/words on unix system. 
***************************************************************

 How to Run :
 1) Run "npm install"
 2) Run "node app.js" and navigate to localhost:3000
 
 Application folder structure and layout as recommended by "Getting MEAN with Mongo, Express,
 Angular, and Node" by Simon Holmes

 Happy Playing.
